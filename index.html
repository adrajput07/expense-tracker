⚙ GitHub Gist Sync Setup

One-time setup:
1. Go to github.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. Give it a name like expense-tracker, tick only the gist scope
3. ⚠ Set an expiration (30 or 90 days) — avoid "No expiration"
4. Paste the token + a passphrase above → Save Settings → click ☁ Push to GitHub
5. Your PAT is AES-256 encrypted before being stored — the raw token is never saved to disk
6. On your other devices, enter the same token + passphrase + copy the Gist ID from here
