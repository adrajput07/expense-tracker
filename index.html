⚙ GitHub Gist Sync Setup

One-time setup:
1. Go to GitHub → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
2. Give it a name like expense-tracker, tick only the gist scope
3. ⚠ Set an expiration (30–90 days) — never use "No expiration" for security
4. Copy & paste the token above → Save Settings → click ☁ Push to GitHub
5. The Gist ID auto-saves. On other devices, enter same token + Gist ID.

🔒 Security note: Your PAT is stored in sessionStorage only (cleared when tab closes). The Gist ID is stored in localStorage. Never share your token.
